명제 논리기초▾

명제: 참/거짓이 명확한 문장. P→Q(함의): P가 참이면 Q도 참. P가 거짓이면 Q와 무관하게 전체는 참. 역(Q→P)과 이(¬P→¬Q)는 원명제와 동치가 아님. 대우(¬Q→¬P)만 동치. 귀류법: P를 거짓이라 가정하면 모순 도출 → P는 참.

종류형태원명제와 동치?
역 (Converse)Q → P아니오
이 (Inverse)¬P → ¬Q아니오
대우 (Contrapositive)¬Q → ¬P예 ✓
예시 문제
Q1 "n이 짝수이면 n²도 짝수이다"의 역·이·대우를 쓰시오.
A. 역: "n²이 짝수이면 n도 짝수". 이: "n이 홀수이면 n²도 홀수". 대우: "n²이 홀수이면 n도 홀수". 원명제와 대우만 동치이고 실제로 모두 참임.
Q2 P=거짓, Q=참일 때 P→Q의 진리값은?
A. 함의 P→Q는 P=거짓이면 Q에 상관없이 참. "거짓이 전제인 약속은 항상 지킨 것"으로 이해.
Q3 √2가 무리수임을 귀류법으로 증명하시오 (개요).
A. √2=p/q (기약분수)라 가정 → 2=p²/q² → p²=2q² → p는 짝수(p=2k) → 4k²=2q² → q²=2k² → q도 짝수. p,q 모두 짝수 → 기약분수 모순 → √2는 무리수 ✓
논리 연산자기초▾

논리 연산은 비트 연산과 직결. AND는 둘 다 참, OR은 하나라도 참, XOR은 홀수 개 참. 함의 P→Q = ¬P∨Q (P가 거짓이거나 Q가 참). 쌍조건 P↔Q = (P→Q)∧(Q→P).

연산기호C++ 비트
ANDP∧Qa & b
ORP∨Qa | b
NOT¬P~a
XORP⊕Qa ^ b
함의P→Q¬P∨Q
예시 문제
Q1 P=참, Q=거짓일 때 P∧Q, P∨Q, P→Q, P⊕Q의 진리값은?
A. P∧Q=거짓, P∨Q=참, P→Q=거짓(참인 P가 거짓 Q를 함의 불가), P⊕Q=참(서로 다름)
Q2 a=5(=101₂), b=3(=011₂)에서 a&b, a|b, a^b를 구하시오.
A. a&b=001₂=1, a|b=111₂=7, a^b=110₂=6
Q3 XOR의 항등원과 역원은 무엇인가? 실제 응용은?
A. 항등원: 0 (a^0=a). 역원: 자기 자신 (a^a=0). 응용: 님 게임 필승 전략(XOR≠0), 배열에서 중복 없는 유일 원소 찾기(모두 XOR하면 쌍들이 0이 됨), 스왑 (a^=b; b^=a; a^=b).
드 모르간 법칙기초▾

AND/OR의 부정을 분배할 때 연산자가 반전됨. 논리·집합·비트 모두 동일하게 적용. 복잡한 조건의 NOT을 구현할 때 자주 사용(루프 탈출 조건 변환, 쿼리 조건 반전 등).

¬(P∧Q) ≡ ¬P∨¬Q ¬(P∨Q) ≡ ¬P∧¬Q 집합: (A∩B)ᶜ=Aᶜ∪Bᶜ (A∪B)ᶜ=Aᶜ∩Bᶜ 비트: ~(a&b)==(~a|~b) ~(a|b)==(~a&~b)
예시 문제
Q1 ¬(P∧¬Q)를 드 모르간으로 단순화하시오.
A. ¬(P∧¬Q) = ¬P∨¬(¬Q) = ¬P∨Q = P→Q (함의와 동치)
Q2 (A∩B)ᶜ = Aᶜ∪Bᶜ를 원소 증명으로 보이시오.
A. x∈(A∩B)ᶜ ⟺ x∉(A∩B) ⟺ ¬(x∈A∧x∈B) ⟺ x∉A∨x∉B ⟺ x∈Aᶜ∨x∈Bᶜ ⟺ x∈Aᶜ∪Bᶜ ✓
Q3 C++ 조건 !(a>0 && b>0)를 드 모르간으로 변환하시오.
A. !(a>0 && b>0) = !(a>0) || !(b>0) = (a<=0 || b<=0)
집합 연산기초▾

집합: 중복 없는 원소의 모음. 비트마스크로 집합을 표현하면 합집합=OR, 교집합=AND, 차집합=AND NOT, 여집합=NOT. 멱집합 2^A: A의 모든 부분집합 집합, |A|=n이면 |2^A|=2^n.

연산표기비트마스크
합집합A∪Ba | b
교집합A∩Ba & b
차집합A\Ba & ~b
여집합Aᶜ=U\A~a (전체 mask 기준)
멱집합2^A0 ~ (1<<n)-1
|A∪B| = |A|+|B|-|A∩B| |2^A| = 2^|A|
예시 문제
Q1 A={1,2,3,4}, B={3,4,5,6}일 때 A∩B, A∪B, A\B를 구하시오.
A. A∩B={3,4}, A∪B={1,2,3,4,5,6}, A\B={1,2}
Q2 |A|=20, |B|=15, |A∩B|=8이면 |A∪B|는?
A. |A∪B| = 20+15-8 = 27
Q3 {0,1,2,3}의 부분집합 {1,3}을 비트마스크로 표현하고, 전체집합 대비 여집합을 구하시오.
A. {1,3} → 비트 1,3번이 켜짐 → 0b1010 = 10. 전체집합=0b1111=15. 여집합=15 & ~10 = 0b0101 = 5 → {0,2}
수학적 귀납법기초▾

①P(1)(또는 P(0)) 기저 사례 증명 ②P(k)가 참이라 가정 후 P(k+1) 증명 → 모든 자연수에서 P(n) 성립. 도미노 원리와 동일. 기저를 여럿 필요로 하거나 이전 여러 단계가 필요하면 강한 귀납법 사용.

예: 1+2+···+n=n(n+1)/2 P(k): 성립 가정 → P(k+1): k(k+1)/2+(k+1)=(k+1)(k+2)/2 ✓
예시 문제
Q1 1+3+5+···+(2n-1)=n²를 귀납법으로 증명하시오.
A. 기저: n=1, 1=1² ✓. 귀납: P(k) 가정: 1+3+···+(2k-1)=k². P(k+1): k²+(2k+1)=k²+2k+1=(k+1)² ✓
Q2 2^n > n (n≥1)을 귀납법으로 증명하시오.
A. 기저: 2^1=2>1 ✓. 귀납: 2^k>k 가정 → 2^(k+1)=2×2^k>2k=k+k≥k+1 (k≥1이므로 k≥1). 따라서 2^(k+1)>k+1 ✓
Q3 Σᵢ₌₁ⁿ i² = n(n+1)(2n+1)/6을 귀납법으로 증명 (귀납 단계만).
A. P(k) 가정 후 P(k+1): k(k+1)(2k+1)/6 + (k+1)² = (k+1)[k(2k+1)/6+(k+1)] = (k+1)[(2k²+k+6k+6)/6] = (k+1)(k+2)(2k+3)/6 ✓
부울 대수기초▾

0과 1로 이루어진 대수 체계. 논리회로와 직결. 흡수 법칙: a+ab=a, a(a+b)=a. 불 합(+)=OR, 불 곱(·)=AND. 카르노 맵으로 부울 식을 최소화해 게이트 수를 줄임. NAND 하나로 AND·OR·NOT 모두 구현 가능(기능적 완전성).

항등: a+0=a, a·1=a 영: a+1=1, a·0=0 멱등: a+a=a, a·a=a 보: a+¬a=1, a·¬a=0 분배: a(b+c)=ab+ac 흡수: a+ab=a, a(a+b)=a 이중부정: ¬(¬a)=a
예시 문제
Q1 a+a·b를 부울 대수로 단순화하시오.
A. a+a·b = a·1+a·b = a(1+b) = a·1 = a (흡수 법칙 직접 적용)
Q2 (A+B)(A+C)를 곱의 합(SOP) 형태로 전개하시오.
A. (A+B)(A+C) = A·A+A·C+B·A+B·C = A+AC+AB+BC = A(1+C+B)+BC = A+BC
Q3 NAND 게이트만으로 NOT을 구현하는 방법은?
A. ¬A = NAND(A,A). 왜냐면 NAND(a,a)=¬(a·a)=¬a. 또한 AND(a,b)=NAND(NAND(a,b),NAND(a,b)), OR(a,b)=NAND(NAND(a,a),NAND(b,b)).
비트 연산 · 비트마스크기초▾

정수를 집합으로 표현: k번째 비트=원소 k 포함 여부. a&(a-1): 최하위 1비트 제거(핵심 트릭). n&(n-1)==0이면 2의 거듭제곱. 부분집합 순회는 SOS DP, 비트마스크 DP의 핵심.

(a>>k)&1          // k번째 비트 확인
a|(1<<k)          // k번째 비트 켜기
a&~(1<<k)         // k번째 비트 끄기
a^(1<<k)          // 토글
a&(a-1)           // 최하위 1비트 끄기
a&(-a)            // 최하위 1비트만 추출
__builtin_popcount(a)  // 켜진 비트 수

// 부분집합 순회 O(2^|mask|)
for(int sub=mask;sub;sub=(sub-1)&mask){ /* sub: mask의 부분집합 */ }
예시 문제
Q1 n=12(=1100₂)에서 n&(n-1)과 n&(-n)을 구하시오.
A. n-1=11(=1011₂). n&(n-1)=1100&1011=1000=8 (최하위 1비트 제거). -n=…10100₂(2의 보수). n&(-n)=0100=4 (최하위 1비트만 추출)
Q2 mask=7(=111₂)의 부분집합 순회에서 방문하는 값은?
A. sub=7→6→5→4→3→2→1 (총 7회, 공집합=0 제외). 모두 방문: 7,6,5,4,3,2,1 → 비어있지 않은 부분집합 7=2³-1개
Q3 n이 2의 거듭제곱인지 O(1)에 확인하는 방법은?
A. n>0 && (n&(n-1))==0. 2의 거듭제곱은 비트가 정확히 하나만 켜짐. n&(n-1)이 그 비트를 끄면 0이 됨. 예) 8=1000, 7=0111, 8&7=0 ✓
강한 귀납법중급▾

일반(약한) 귀납법: P(k)→P(k+1). 강한 귀납법: P(1)∧P(2)∧···∧P(k)→P(k+1). 이전 모든 단계를 쓸 수 있어 더 강력. 소인수분해 유일성, 피보나치 점화식, 게임 이론 Grundy 수 증명에 필수. 구조적 귀납법은 트리·식 등 재귀 구조에 적용.

예시 문제
Q1 "n>1인 모든 정수는 소수의 곱으로 표현된다"를 강한 귀납법으로 증명 (개요).
A. 기저: n=2, 소수 2 = 소수 곱 ✓. 귀납: 2~k까지 성립 가정. k+1이 소수면 자명. 합성수면 k+1=a×b(2≤a,b≤k) → a,b 각각 소수 곱 표현 가능(강한 귀납 가정) → k+1도 소수 곱 ✓. 약한 귀납으로는 a,b가 k이하임을 활용 불가.
Q2 피보나치 Fₙ≤2^n을 강한 귀납법으로 증명하시오.
A. 기저: F₁=1≤2, F₂=1≤4 ✓. 귀납: F(k)≤2^k, F(k-1)≤2^(k-1) 가정. F(k+1)=F(k)+F(k-1)≤2^k+2^(k-1)=2^(k-1)×3<2^(k-1)×4=2^(k+1) ✓
Q3 님 게임에서 강한 귀납법이 필요한 이유는?
A. Grundy(s)=mex{Grundy(next)} 정의에서 다음 상태가 여러 단계 이전일 수 있음. 약한 귀납: P(k-1)→P(k)만 사용 가능. 강한 귀납: 모든 이전 상태의 Grundy 값이 정확히 계산됐다고 가정 → P(k) 증명 가능.
동치 · 편순서 관계중급▾

관계 R⊆A×A. 반사: aRa 항상 성립. 대칭: aRb→bRa. 추이: aRb∧bRc→aRc. 반대칭: aRb∧bRa→a=b. 동치 관계는 집합을 동치류로 분할(서로소 분할). 편순서의 Hasse 다이어그램으로 시각화.

종류성질예시
동치 관계반사+대칭+추이mod m 합동, 같은 SCC
편순서 관계반사+반대칭+추이약수 관계 |, 집합 포함 ⊆
전순서 관계편순서+임의 쌍 비교 가능정수의 ≤, 사전순
예시 문제
Q1 정수에서 "mod 3 합동(≡₃)"이 동치 관계임을 세 성질로 확인하시오.
A. 반사: a≡a(mod 3) ✓. 대칭: a≡b→b≡a(mod 3) ✓. 추이: a≡b, b≡c→a≡c(mod 3) ✓ → 동치 관계. 동치류: {…,-3,0,3,6,…}, {…,-2,1,4,7,…}, {…,-1,2,5,8,…}
Q2 자연수에서 "약수 관계 a|b"가 편순서임을 확인하시오.
A. 반사: a|a ✓. 반대칭: a|b∧b|a → a=b(양수) ✓. 추이: a|b∧b|c → a|c ✓. 하지만 임의 쌍에서 2|3은 성립 안 하므로 전순서가 아닌 편순서.
Q3 그래프의 SCC와 동치 관계의 연결을 설명하시오.
A. 방향 그래프에서 "u에서 v에 도달 가능하고 v에서 u에도 도달 가능"이 동치 관계. 반사(자기 자신 도달), 대칭(정의상), 추이(경로 합성). 각 동치류 = SCC.
술어 논리 · 한정 기호중급▾

술어 P(x): x에 따라 참/거짓이 달라지는 명제 함수. ∀(전칭): 모든 x, ∃(존재): 어떤 x. 부정 시 한정 기호가 반전되고 술어도 부정됨(드 모르간의 한정 기호 버전). 유일 존재 ∃!x P(x) = ∃x(P(x)∧∀y(P(y)→y=x)).

∀x P(x): 모든 x에서 P(x) 참 ∃x P(x): P(x)가 참인 x 존재 ¬(∀x P(x)) ≡ ∃x ¬P(x) ¬(∃x P(x)) ≡ ∀x ¬P(x) ∀x∃y P(x,y) ≢ ∃y∀x P(x,y) [순서 중요]
예시 문제
Q1 "모든 소수는 홀수이다"를 술어 논리로 쓰고 참/거짓을 판단하시오.
A. ∀x(Prime(x)→Odd(x)). 거짓. 반례: 2는 소수이나 짝수. 즉 ¬(∀x P(x))=∃x¬P(x) → 2라는 반례가 존재.
Q2 ¬(∃x(x>0 ∧ x²<0))를 단순화하시오.
A. ¬∃x P(x) = ∀x¬P(x) = ∀x¬(x>0∧x²<0) = ∀x(x≤0∨x²≥0). 실제로 모든 실수에서 x²≥0이므로 이 명제는 참.
Q3 "모든 x에 대해 어떤 y>x가 존재한다"와 "어떤 y가 모든 x보다 크다"의 차이를 설명하시오.
A. ∀x∃y(y>x): 정수에서 참 (y=x+1). ∃y∀x(y>x): 정수에서 거짓 (최대 정수 없음). ∀x∃y는 y가 x에 따라 달라질 수 있음, ∃y∀x는 고정된 하나의 y가 모든 x에 성립해야 함.
